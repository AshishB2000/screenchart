Screenchart / Permission

Screenchart can't record the screen

macOS is blocking screen capture. If Screenchart is already listed in Screen Recording, that entry is from a previous version and no longer matches this build — remove it and let macOS ask again.

  1. 1 Click Open System Settings below (jumps to Screen & System Audio Recording).
  2. 2 If Screenchart is in the list, select it and click the – (minus) button to remove the old entry.
  3. 3 Quit Screenchart completely — ⌘Q.
  4. 4 Reopen Screenchart and press New capture. macOS will ask to record the screen — click Allow.

Screenshot any chart or table and get a clear, plain-English read — powered by your own AI key.

Troubleshooting
Capture comes back black
Grant Screen Recording permission in System Settings.
"Key was rejected"
Re-paste your API key under Settings → Execution mode.
A number looks wrong
Re-capture tighter around the table so text stays crisp.
